# Gestor de Libros - API REST

API REST desarrollada con **Spring Boot**.  
Implementa un **CRUD completo** siguiendo **arquitectura en capas** y utilizando **almacenamiento en memoria**.

---

## Funcionalidades

- Crear libros
- Obtener libros (por ID y listado)
- Actualizar libros
- Eliminar libros
- Manejo de errores con códigos HTTP correctos

---

## Arquitectura

- Controller
- Service
- Repository
- Excepciones personalizadas

---

## Tecnologías

- Java
- Spring Boot
- Maven
